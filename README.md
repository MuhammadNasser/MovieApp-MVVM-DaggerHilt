# MovieApp-MVVM-DaggerHilt
simple app, where you can see how to use dagger-hilt, retrofit2, mvvm, livedata and so much more...

# Features
- MVVM
- Dagger-Hilt
- LiveData
- Retrofit2
- Movie Api
- Timber
- Room Database
- Fragment navigation with bottom navbar
- BindingAdapters
